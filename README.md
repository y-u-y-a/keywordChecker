[ユーザーの使い方]
- [記事作成]
・キーワードの投入
・H1、H2、のおすすめキーワードと、含んだ方がいいキーワード（サジェスト）を抽出
- [チェック]
・URLを投入
・キーワードの網羅率によって、点数が出てくる
・含めた方がいいキーワードがわかる

- [記事作成]
・サジェストキーワード（3語）の網羅性
・サジェストキーワードの検索ボリュームが多いキーワードのH2/H3タグへの投入率
・狙ったキーワードの上位10サイトの
・タイトル
・URL
・文字数
・月間検索ボリューム
・顕在ニーズ含む、月間検索ボリューム（2語+3語キーワードの全検索ボリューム）
・平均文字数
・平均見出し数
・想定アクセス数
